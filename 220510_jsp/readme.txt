Server - Client 구조에서는 n. (효율적)

	2. Server : 연산
	3. Server -> Client : html 파일을 보냄
	4. Client : html 파일을 해석해 브라우저에 띄움

App(-lication : 응용 프로그램)
Web = Server Application
